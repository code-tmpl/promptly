# Promptly

A native macOS teleprompter that uses your MacBook's notch area to keep your script right at the camera — so you maintain natural eye contact while presenting.

## Features

- **Notch-based prompting** — Text appears right at the camera for natural eye contact
- **Voice-activated scrolling** — Script follows your speaking pace
- **Screen-share invisible** — Only you can see the prompter
- **Floating window mode** — Place anywhere, resize freely
- **Pause on hover** — Instant control without breaking flow
- **Speed control** — Keyboard shortcuts for real-time adjustments
- **Local & private** — All scripts stored on your Mac, no cloud

## Requirements

- macOS 14.7+ (Sonoma)
- Intel or Apple Silicon Mac
- Notch-equipped MacBook recommended (floating mode works on any Mac)

## Build

```bash
xcodebuild -project Promptly.xcodeproj -scheme Promptly build
```
